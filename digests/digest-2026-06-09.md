## 📄 Research Papers — Top Picks
- **From Conflict to Consensus: Boosting Medical Reasoning via Multi-Round Agentic RAG** — Wu et al. · https://arxiv.org/abs/2603.03292
  MA-RAG treats disagreement among candidate answers as a signal, iteratively turning semantic conflict into new retrieval queries while pruning long-context noise in the reasoning history. Across 7 medical QA benchmarks it adds +6.8 avg accuracy over the backbone, beating inference-time-scaling and RAG baselines. **Why it matters:** Shows test-time agentic refinement, not bigger models, can close reliability gaps in high-stakes domains.
- **RECON: Reasoning with Condensation for Efficient Retrieval-Augmented Generation** — Xu et al. · https://arxiv.org/abs/2510.10448
  RECON inserts a trained, frozen observation compressor into a multi-turn RL search-agent loop so accumulating tool outputs don't inflate context. Integrated into Search-R1 it cuts context 35%, speeds training 5.4% and inference 30.9%, and lifts exact-match up to +14.5% on multi-hop QA. **Why it matters:** Learned observation compression is becoming a core building block for scalable, tool-using RL agents.
- **Syll: Open-Source Personal Automation with Cross-Surface Execution** — Zhang et al. · https://arxiv.org/abs/2606.07594
  Syll is a self-hosted multimodal agent harness unifying MCP/API tools, CLI, and visual GUI control, where users teach procedures by demonstration that compile into reusable skills. Execution returns logs, keyframes, and approval checkpoints, validated on Photoshop, Audition, Finder, and games. **Why it matters:** A teachable, auditable, open harness for cross-surface computer-use agents users can inspect and extend.
- **Anything2Skill: Compiling External Knowledge into Reusable Skills for Agents** — Pan et al. · https://arxiv.org/abs/2606.09316
  Anything2Skill compiles heterogeneous knowledge (manuals, logs, trajectories) into structured, versioned skill contracts stored in a persistent SkillBank, so agents retrieve both declarative passages and procedural skills. On qsv and GitHub-CLI it hits 98.9% and 94.1% success, far above RAG-only. **Why it matters:** Moves RAG from knowledge access toward reusable capability, a key step for durable agent skill libraries.
- **Resilient Write: A Six-Layer Durable Write Surface for LLM Coding Agents** — Agyemang et al. · https://arxiv.org/abs/2604.10842
  An MCP server interposing six orthogonal layers (risk scoring, atomic writes, resume-safe chunking, typed errors, scratchpad storage, handoff envelopes) between coding agents and the filesystem. A 186-test suite shows 5x faster recovery and 13x better self-correction versus naive baselines. **Why it matters:** Hardens the brittle MCP write path that silently loses drafts and wastes tokens during real coding-agent sessions.
- **Goal-Oriented Reasoning for RAG-based Memory in Conversational Agentic LLM Systems** — Liang et al. · https://arxiv.org/abs/2605.12213
  Goal-Mem does explicit backward chaining from the user's request, decomposing it into atomic subgoals and retrieving memory to satisfy each, formalized in a Natural Language Logic system. Against nine memory baselines it consistently wins, especially on multi-hop and implicit-inference tasks. **Why it matters:** Reasoning-guided retrieval beats raw semantic-similarity memory for long-horizon conversational agents.

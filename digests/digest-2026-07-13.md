## 📄 Research Papers — Top Picks
- **LongMedBench: Benchmarking Medical Agents for Long-Horizon Clinical Decision-Making** — Yanzhen Chen et al. · https://arxiv.org/abs/2607.09322
  An EHR-based benchmark for long-horizon clinical decisions built from MIMIC-IV (335 patients, ~20 visits each), with three suites: fact QA, temporal reasoning, and long-horizon decision-making. Experiments show LLM agents use explicit timestamps well but struggle with implicit time inference; RAG and agent-memory help retrieval yet decision quality still hinges on immediate context. **Why it matters:** exposes that current agent-memory systems don't yet close the gap on longitudinal clinical reasoning.
- **Shared Selective Persistent Memory for Agentic LLM Systems** — Sanjana Pedada et al. · https://arxiv.org/abs/2607.09493
  An architecture that persists only four reusable context categories (task specs, data schemas, tool configs, output constraints) across agent sessions while discarding session-specific reasoning traces, and shares them across users via RBAC. It reaches 96% task completion vs 79% without memory and 71% with full history; a zero-token refresh cuts recurring-update time 14x. Naive full-history persistence actively degrades performance by biasing the agent with stale traces. **Why it matters:** concrete evidence that selective, shareable memory beats both no-memory and full-history for agentic coding systems.
- **AgentKGV: Agentic LLM-RAG Framework with Two-Stage Training for the Fact Verification of Knowledge Graphs** — Yumin Heo et al. · https://arxiv.org/abs/2607.09092
  An agentic LLM-RAG framework for knowledge-graph fact verification with dynamic routing and iterative query rewriting, plus two-stage training (distillation SFT then trajectory-level GRPO). It improves macro-F1 over single-turn RAG by 5.5pp, and +9.4pp with the two-stage training; GRPO cuts average search calls from 3.24 to 1.63 without lowering accuracy. **Why it matters:** shows RL-tuned search policies make verification agents both more accurate and cheaper at industrial scale.
- **Eluna: An Agentic LLM System for Automating Warehouse Operations with Reasoning and Task Execution** — Ning Liu et al. · https://arxiv.org/abs/2607.08960
  A production-deployed agentic system that executes warehouse SOPs encoded as DAGs with progressive disclosure and parallel sub-agents holding persistent code execution. Asymmetric episodic distillation lets a small fine-tuned student match or beat its teacher, reaching 94% expert agreement on a ticket-processing application. **Why it matters:** a deployed template for enforcing procedural compliance in real-world multi-agent operations under latency constraints.
- **QAgent: An LLM-based Multi-Agent System for Autonomous OpenQASM programming** — Zhenxiao Fu et al. · https://arxiv.org/abs/2508.20134
  The first autonomous multi-agent framework for end-to-end OpenQASM quantum-circuit code, unifying schema-aware planning, RAG-driven synthesis, and hardware-aware calibration with iterative execution feedback. It improves Pass@1 by 47–70% on single kernels and exceeds 88% on multi-kernel workflows, holding near-unit fidelity under hardware frequency drift where SDK-based LLM methods degrade. **Why it matters:** demonstrates agent pipelines can reliably generate low-level, hardware-specific code that plain LLM+SDK approaches cannot.
- **Agora: Enhancing LLM Agent Reasoning Via Auction-Based Task Allocation** — Kaiji Zhou et al. · https://arxiv.org/abs/2607.09600
  A framework that treats reasoning steps as tradeable items and uses an incentive-compatible auction to route each step to the most capable expert model/tool by rectified competence, rather than the most overconfident one. Across five benchmarks it beats single-model, routing, and cascade baselines while exposing a cost-quality trade-off through one auction parameter. **Why it matters:** a principled orchestration mechanism for multi-model agent systems that controls cost versus quality.
- **Task-Specific Multimodal Question Answering Agents via Confidence Calibration and Incremental Reasoning for QANTA 2026** — Nirjhar Das, Md. Al-Mamun Provath · https://arxiv.org/abs/2607.09623
  A task-specific two-agent architecture for multimodal quizbowl: a confidence-calibrated Tossup agent that decides when to answer under uncertainty, and a leadin-aware Bonus agent for exact answer selection. Using only hosted GPT-4.1-mini/GPT-4.1 with no retrieval or ensembles, it won the challenge with the top leaderboard score of 0.402. **Why it matters:** lightweight, calibrated reasoning policies can top heavier pipelines under real efficiency constraints.
- **Mach-Mind-4-Flash Technical Report** — Foundation Model Team · https://arxiv.org/abs/2607.09375
  A 35B Mixture-of-Experts agentic model (3B activated) that, through post-training only, matches or beats 100B-class models. The pipeline combines a unified RL/OPD infrastructure, parallel domain RL experts fused via Multi-Teacher On-Policy Distillation, and Hybrid Median-length Policy Optimization that compresses reasoning chains 19–46% at ≤0.7pp accuracy loss; it scores 92.7 on AIME'26 and 75.8 on BFCL-v4. **Why it matters:** shows agentic RL plus distillation can deliver frontier tool-use performance at a fraction of activated size and inference cost.
- **GrAInS: Gradient-based Attribution for Inference-Time Steering of LLMs and VLMs** — Duy Nguyen et al. · https://arxiv.org/abs/2507.18043
  An inference-time steering method that uses contrastive Integrated-Gradients attribution to find the most influential tokens and build directional steering vectors, adjusting hidden activations without retraining. It outperforms fine-tuning and prior steering baselines: +13.2% on TruthfulQA (Llama-3.1-8B), lower hallucination on MMHal-Bench, and +8.1% alignment win rate on SPA-VL. **Why it matters:** a cheaper, interpretable alternative to fine-tuning for controlling behavior across language and vision-language models.
- **Fictional Worldbuilding: Multi-Agent LLM Collaboration with Hierarchical Context Compression and Iterative Review** — Jingbo Chen et al. · https://arxiv.org/abs/2607.09403
  AutoWorldBuilder, a multi-agent system for coherent fictional worlds, combines a conflict-detecting concept network, a DAG batch scheduler, four-layer context compression (~90% token reduction), and Auditor agents that lift proposal pass rates from 42% to over 85%. It reaches a 95% success rate, generating 56–103 self-consistent concepts per world in 18–31 minutes. **Why it matters:** its layer-as-budget compression and generation/review separation transfer to knowledge-intensive multi-agent applications broadly.
- **Evaluating Retrieval-Augmented Generation vs. Long-Context Input for Clinical Reasoning over EHRs** — Skatje Myers et al. · https://arxiv.org/abs/2508.14817
  A comparison of targeted retrieval against long-context prompting on three EHR tasks across GPT-5.4-mini, Mistral Medium 3, and DeepSeek V3.1. RAG matches or beats long-context using under 8K tokens for imaging extraction and antibiotic timelines, while diagnosis generation stays largely flat across methods. **Why it matters:** RAG remains a token-efficient, competitive approach for clinical reasoning even as long-context models improve.
- **REAL: REtrieval-reAsoning and Logic-constructed Attention Behaviors for Long-Context KV Cache Compression** — Mengjie Li et al. · https://arxiv.org/abs/2508.15806
  A KV-cache eviction method that analyzes attention-head behavior in both success and failure cases via an Attention Behavior Matrix, strengthening valid reasoning pathways while suppressing bias and distraction. On LongBench v2 it matches the strongest baseline (HeadKV-R2) while using 32x less cache space. **Why it matters:** failure-aware head analysis unlocks far cheaper long-context inference.
- **GATS: Graph-Augmented Tree Search with Layered World Models for Efficient Agent Planning** — Maureese Williams, Dymitr Nowicki · https://arxiv.org/abs/2607.08894
  A planning framework pairing UCB1 tree search with a three-layer world model (exact symbolic matching, execution-log statistics, LLM fallback) to eliminate LLM calls at inference. It reaches 100% success on a 12-scenario stress test where LATS drops to 88.9% and ReAct to 23.9%, producing deterministic zero-variance plans with zero LLM calls per task. **Why it matters:** learned world models can replace costly, stochastic LLM-guided exploration for agent planning.
- **Long-Horizon-Terminal-Bench: Testing the Limits of Agents on Long-Horizon Terminal Tasks with Dense Reward-Based Grading** — Zongxia Li et al. · https://arxiv.org/abs/2607.08964
  A terminal benchmark of 46 long-horizon tasks across nine categories, decomposed into fine-grained graded subtasks for dense/partial rewards; tasks average ~231 episodes and ~85 minutes each. Even the strongest of 15 frontier models reaches only 15.2% pass@1 at a 0.95 reward threshold, with a 4.3% mean across models. **Why it matters:** quantifies how far agents remain from reliable long-horizon terminal work while providing partial-credit signals for progress.
- **Neuro-Agentic Control: A Deep Learning-based LLM-Powered Agentic AI Framework for Controlling Security Controls** — Saroj Gopali et al. · https://arxiv.org/abs/2607.09076
  A framework coupling an LLM planner with a time-series foundation model (TimesFM) and a "Counterfactual Physics Injection" step that simulates proposed interventions in latent space before actuation to reject hallucinatory or unsafe actions. On the SWaT industrial dataset it prevented 33.3% of breaches with zero physically invalid actions, beating LSTM and TCN baselines. **Why it matters:** shows foundation-model "sentinels" can safely gate agentic control in critical infrastructure.

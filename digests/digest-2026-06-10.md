## 📄 Research Papers — Top Picks
- **TabClaw: An Interactive and Self-Evolving Agent for Spreadsheet Manipulation and Table Reasoning** — Ming et al. · https://arxiv.org/abs/2606.10316
  TabClaw is an open-source interactive agent for spreadsheet manipulation and table reasoning: users upload CSV/Excel files and issue natural-language requests. It targets known LLM-agent weaknesses — opaque intermediate decisions, implicit assumptions, weak multi-table comparison — and self-evolves by adapting workflows to user preferences. **Why it matters:** Transparent, preference-learning table agents attack the most common real-world structured-data workload.
- **ASA: Backbone-Training-Free Representation Engineering for Tool-Calling Agents** — Youj et al. · https://arxiv.org/abs/2602.04935
  Identifies a "Lazy Agent" failure mode where tool necessity is nearly perfectly decodable from mid-layer activations yet the model stays conservative about entering tool mode — a representation-behavior gap. The proposed Activation Steering Adapter (ASA) closes the gap training-free at inference time, avoiding both fragile prompt engineering and costly continual fine-tuning. **Why it matters:** A cheap, deployable lever to make tool-calling agents actually use their tools under interface drift.
- **One Token per Multimodal Evidence: Latent Memory for Resource-Constrained QA** — Zhi et al. · https://huggingface.co/papers/2606.10572
  Latent Memory replaces each raw text/image evidence item in external memory with a single high-dimensional latent token produced by a small compressor LLM/VLM. This slashes token consumption and storage pressure versus passing retrieved raw text and images into the generator. **Why it matters:** One-token-per-evidence retrieval could make RAG affordable on resource-constrained and edge deployments.
- **Kwai Keye-VL-2.0 Technical Report** — Kwai · https://huggingface.co/papers/2606.10651
  Keye-VL-2.0-30B-A3B is an open-source MoE multimodal foundation model for long-video understanding and agentic intelligence. It is the first to adapt DeepSeek Sparse Attention to GQA-based multimodal architectures, enabling lossless 256K context over hour-level videos, backed by scalable video I/O and heterogeneous ViT-LM parallelism. **Why it matters:** Open hour-scale video understanding with 256K context expands what multimodal agents can ingest.
- **WebChallenger: A Reliable and Efficient Generalist Web Agent** — Jayo et al. · https://arxiv.org/abs/2606.10423
  Argues the web-agent gap stems from architecture, not model capability, and replicates three human advantages: selective attention to relevant page regions, persistent memory of website structure, and procedural fluency with common interaction patterns. The result is a generalist web agent that avoids dependence on expensive proprietary reasoning models. **Why it matters:** Architecture-over-scale web agents make repetitive browsing automation economically viable.
- **Pushing the Limits of LLM Tool Calling via Experiential Knowledge Integration and Activation** — Yupu et al. · https://arxiv.org/abs/2606.10875
  A systematic study of how knowledge affects multi-step tool use across acquisition, activation, and internalization stages. Finds simple instance-level experiential knowledge already gives strong, reliable gains, while abstract intent-level knowledge helps little. **Why it matters:** Concrete guidance on what kind of "experience" actually improves agent tool reliability.
- **Decentralized Multi-Agent Systems with Shared Context (DeLM)** — Yuzh et al. · https://arxiv.org/abs/2606.10662
  DeLM removes the centralized orchestrator bottleneck in multi-agent systems: parallel agents asynchronously claim subtasks from a task queue, read a shared verified context, reason locally, and write back compact progress. **Why it matters:** Decentralized coordination is a credible path to scaling test-time multi-agent reasoning past the controller bottleneck.
- **Trace Only What You Need: Structure-Aware On-Demand Hypergraph Memory for Long-Document QA (DocTrace)** — Xian et al. · https://arxiv.org/abs/2606.10921
  DocTrace is a multi-agent RAG framework for long-document QA that builds hypergraph memory on demand, exploits original document structure, and reuses historical reasoning experience — avoiding costly query-agnostic preprocessing. **Why it matters:** On-demand structured memory cuts the indexing tax that makes structured RAG impractical at scale.
- **Infini Memory: Maintainable Topic Documents for Long-Term LLM Agent Memory** — Suoz et al. · https://arxiv.org/abs/2606.10677
  Treats agent memory as topic-structured documents: each topic doc is a semantic unit collecting related evidence, preserving metadata, and revising facts over time, with new observations staged in a buffer and periodically consolidated. **Why it matters:** Maintainable, revisable text memory directly addresses fact drift in long-running agents.
- **Skill-RAG: Failure-State-Aware Retrieval Augmentation via Hidden-State Probing and Skill Routing** — Kai et al. · https://arxiv.org/abs/2604.15771
  Observes that persistent retrieval failures often come from query-evidence misalignment, not missing evidence, and that retry-based adaptive retrieval never diagnoses this. Skill-RAG couples lightweight hidden-state probing with skill routing to repair the alignment gap. **Why it matters:** Diagnosing retrieval failure beats blind retries — a practical upgrade for production RAG.
- **HIPIF: Hierarchical Planning and Information Folding for Long-Horizon LLM Agent Learning** — Junc et al. · https://arxiv.org/abs/2606.10507
  Targets long-context interference in multi-turn agents, where growing histories weaken global task-state tracking. Combines hierarchical planning with information folding to compress history while preserving decision-relevant state. **Why it matters:** Long-horizon degradation is the top failure mode of deployed agents; folding history is a direct countermeasure.
- **IntentKV: Cross-Turn Intent-Aware KV Cache Pruning for Agent Inference** — Junj et al. · https://arxiv.org/abs/2606.09916
  Notes that KV cache, not parameter compute, dominates serving cost for long-horizon agents whose trajectories balloon with tool calls and search results. IntentKV learns to prune KV with a session-level QueryMemory of cross-turn intent while keeping the base LLM frozen and staying composable with prefix caching. **Why it matters:** Intent-aware KV pruning attacks the real serving bottleneck of agent inference.
- **SkillResolve-Bench: Measuring and Resolving Same-Capability Ambiguity in Agent Skill Retrieval** — Jian et al. · https://arxiv.org/abs/2606.10388
  Studies a subtle skill-retrieval failure: the retriever finds the right capability family but the wrong same-capability sibling — one with stale resources, missing preconditions, or wrong procedures. SkillResolve-Bench 1.0 makes this execution-risk retrieval failure auditable. **Why it matters:** As agent skill libraries grow, picking the *right version* of a capability becomes the new retrieval problem.
- **Divide and Cooperate: Role-Decomposed Multi-Agent LLM Training with Cross-Agent Learning Signals** — Jaew et al. · https://arxiv.org/abs/2606.10684
  DAC decomposes evidence acquisition and answer generation into separate cooperating policies, fixing the credit-assignment problem where a good retrieval action is penalized because generation failed. **Why it matters:** Role decomposition with cross-agent signals makes multi-step agent RL training cleaner and more sample-efficient.
- **Role-Agent: Bootstrapping LLM Agents via Dual-Role Evolution** — Xuco et al. · https://huggingface.co/papers/2606.10917
  One LLM plays both agent and environment: World-In-Agent predicts future states after each action and uses prediction-actual alignment as a process signal, enabling bootstrapped co-evolution without static training environments. **Why it matters:** Self-generated environments sidestep the training-data bottleneck for agent learning.
